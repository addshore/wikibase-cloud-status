Wikibase cloud status

Showing you the last 7 days of data collected every 1 minuite for the wikibase.cloud platform from a series of checks

You may also be interested in https://wmde.github.io/wikibase-cloud-status/ for the official platform status.
Code and data by addshore

General

  • Each service is checked every 1 minuite, and a response time is recorded.
  • Each service is checked for a 200 status code.
  • elastic is also checked to be returning a valid result.
  • Green indicates OK, red is bad!
